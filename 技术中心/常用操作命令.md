### **Mac启动台应用程序图标排列数量**

1、打开终端（terminal）

2、设置一行多少个图标

```
defaults write com.apple.dock springboard-columns -int 10
```
3、设置多少行显示图标数量
```
defaults write com.apple.dock springboard-rows -int 7
```
4、重置Launchpad
```
defaults write com.apple.dock ResetLaunchPad -bool TRUE
```
5、重启Docker
```
killall Dock
```
